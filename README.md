# sweep

## Install
```bash
npm i -g @techgreedy/sweep
```

## Usage
```shell
$ sweep -h
Usage: sweep [options]

A tool to get transfer amount for precommits and provecommits

Options:
  -V, --version            output the version number
  --from <height_or_date>  Height or Date as start range (inclusive) (default: "0")
  --to <height_or_date>    Height or Date as end range (exclusive) (default: "999999999")
  --miner <address>
  -h, --help               display help for command
```
